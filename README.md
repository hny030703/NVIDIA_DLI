#6. DLI docker Install

* Headless Mode
  Headless Mode인 경우, Jetson은 모니터나 키보드가 필요하지 않다.
  gui 없이 Jetson의 화면은 검정색이 되며, 이를 '헤드리스 모드'라고 한다.
  Jetson Nano가 처음 부팅될 때 스크립트를 통해 사용자는 사용자 이름과 암호, 언어, 시간대 등을 설정할 수 있다.
